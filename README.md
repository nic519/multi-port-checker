
## 解决的问题
在多端口的模式下：
- 检查代理端口是否可用
- 检查多个代理到底有多少个不同的IP（机场是否良心）


## 安装依赖

```bash
yarn install
```

## 使用方法

### 1. 直接运行
```bash
yarn start
```

### 2. 开发模式（监听文件变化）
```bash
yarn dev
```

### 3. 编译后运行
```bash
npm run build
node dist/view_ip.js
```

## 配置文件

在`main()`函数中修改配置文件路径
- 支持读取本地文件
- 支持读取远程文件
